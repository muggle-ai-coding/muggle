## 🎯 你將學到什麼

在本教學中，你將學習如何：
- 在 Google 試算表中設定 Apps Script
- 撰寫簡單的程式碼來自動化資料收集
- 排程讓自動化程式定期執行

## 📋 問題描述

你需要每天從表單或外部來源收集資料並整理到 Google 試算表中。手動處理既無聊又耗時！

## ✨ 解決方案

我們將建立一個自動化系統：
1. 自動收集資料
2. 在試算表中整理資料
3. 按排程執行（每天、每小時等）

## 🚀 詳細步驟

### 步驟 1：開啟 Google 試算表

1. 前往 [Google 試算表](https://sheets.google.com)
2. 建立新的試算表或開啟現有的試算表
3. 將試算表命名為「我的第一個自動化」

### 步驟 2：開啟 Apps Script 編輯器

1. 點擊選單中的 **擴充功能**
2. 選擇 **Apps Script**
3. 新分頁會開啟腳本編輯器

![Apps Script 選單](https://via.placeholder.com/800x400?text=截圖位置)

### 步驟 3：撰寫你的第一段程式碼

刪除預設程式碼並貼上以下內容：

```javascript
function myFirstAutomation() {
  // 取得目前的試算表
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  
  // 取得目前日期和時間
  var now = new Date();
  
  // 新增一列資料包含時間戳記
  sheet.appendRow([now, "自動化正在運作！"]);
  
  Logger.log("資料已成功新增！");
}
```

**這段程式碼做了什麼？**
- 取得你目前的試算表
- 取得目前的日期和時間
- 新增一列包含時間戳記和訊息的資料

### 步驟 4：測試你的自動化

1. 點擊上方的 **執行** 按鈕（▶️）
2. 系統會要求你授權腳本 - 點擊 **審查權限**
3. 選擇你的 Google 帳號
4. 點擊 **允許**
5. 檢查你的試算表 - 你應該會看到新的一列資料！

### 步驟 5：設定自動執行

1. 點擊左側邊欄的 **觸發條件** 圖示（⏰）
2. 點擊 **+ 新增觸發條件**
3. 設定：
   - 選擇函式：`myFirstAutomation`
   - 事件來源：`時間驅動`
   - 類型：`日計時器`
   - 時段：選擇你偏好的時間
4. 點擊 **儲存**

🎉 **恭喜！** 你的自動化現在會每天自動執行了！

## 🆘 常見問題

**問題：** 「需要授權」錯誤  
**解決方案：** 仔細遵循授權步驟。Google 只是在確認你批准這個腳本。

**問題：** 腳本沒有自動執行  
**解決方案：** 檢查觸發條件設定。確保建立觸發條件後有儲存。

**問題：** 程式碼錯誤  
**解決方案：** 複製錯誤訊息並詢問 AI 助理！
